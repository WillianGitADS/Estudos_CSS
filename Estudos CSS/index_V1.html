Grupamento de Seletores

Podemos aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os sleetores utilizando uma birgula(,).

Assim.não precimos ficar copiando e colando os mesmos códigos para todos, nõs podemos fazer um só bloco de CSS e reutilizar as mesmas regras!
